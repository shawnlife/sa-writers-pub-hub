How to use this tool

This tool replaces manually checking ~400 Substack writers' pages each digest. The database updates every morning with whatever each writer posted the day before — generating a digest is nearly instant.

⚠ Don't include today's date. The database only syncs overnight, so today's posts won't be in there yet — pick dates up to yesterday.

Writers tab

  • Browse, search, and filter all writers. Click any row to edit that writer.
  • Use + Add writer to add someone new. Set their Substack profile link (format: substack.com/@username) and their Publication RSS feeds (format: username.substack.com/feed, comma-separate if they have multiple publications).
  • Last post colours: green = posted in the last 30 days, amber = 31–90 days ago, red = over 90 days ago, grey = no feed or no posts found.

Digest tab

  • Pick a date range on the left and a digest type on the right, then click Generate digest.
  • Writers who published but have no category will appear in the Uncategorised writers panel — assign a category and save. Changes write back to the sheet immediately.
  • Click Create digest to build the formatted digest. Review and edit it inline, then either Copy for Substack (paste straight into a new post) or Open in Google Docs to access the archived copy.
