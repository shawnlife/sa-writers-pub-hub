Introduction & need help?

This tool replaces manually checking ~400 Substack writers' pages each digest. There's a database behind the scenes that gets populated every morning with whatever each writer has posted, so generating a digest here is nearly instant.

⚠ Don't include today's date in your digest. The database only updates each morning with the previous day's articles, so today's posts won't be in there yet — only pick dates up to yesterday.

Add / update a writer

  • + Add writer — add someone new with their name, Substack link, and category.
  • Update writer — pick an existing writer to fix their name, link, or category. If you rename someone, their past articles are relinked automatically.

Generate a digest

  • Pick a date range and (optionally) a single category, then click Generate digest.
  • Any writer who published but has no category yet will show up in the Uncategorised writers panel — assign one and click Save to sheet. This updates both the writer list and the article database.

Creating the digest

  • Create digest in Google Doc writes a formatted copy into the running Google Doc (handy as an archive/working copy).
  • Copy digest for Substack copies a ready-to-paste version: writer names in bold, article titles as plain links, category photos with credits where available. Paste it straight into a new Substack post.
